# CI GitHub Action

The CI GitHub Action is a planned Session V2.7/V2.x integration surface. Until it ships, treat this page as the intended contract rather than a live API.

## Intended behavior

On pull requests, Archon should:

- detect changed Solidity files,
- run configured gas snapshots,
- compare against the base branch,
- post a concise PR comment,
- fail only when configured thresholds are exceeded,
- link back to the full Archon report.

## Example planned config

```yaml
name: Archon Gas Guard
on: [pull_request]
jobs:
  gas:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: archon/gas-guard-action@v1
        with:
          command: forge snapshot
          max-regression-bps: 300
```

This package name is illustrative until the action is published.
