# FAQ

## Is Archon a formal audit?

No. Archon is an AI-assisted review and evidence product. It can find issues and produce useful reports, but high-value launches still need human review and production monitoring.

## Why does Archon care about Mantle gas separately?

Because total user cost can include L2 execution and L1 data availability components. Optimizing only execution can miss savings from smaller calldata or logs.

## Are proof links always available?

No. Proof links should appear only when a real proof transaction or registry record exists. Pending, failed, or unavailable proofs must be labeled honestly.

## Can I paste private keys into Archon?

No. Never paste private keys, seed phrases, deployer keys, or production secrets into Archon.

## Why are some API pages labeled planned?

Because the public platform API is scheduled for a later session. The docs keep the IA ready without pretending unfinished endpoints are live.
