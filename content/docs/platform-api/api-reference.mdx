# API reference

The public API reference is planned for Session V2.7. Until then, this page documents stable product expectations rather than a published external contract.

## Current internal surfaces

Archon already uses app routes for product data such as gas leaderboard and authenticated app workflows. External consumers should not treat these internal routes as stable unless they are documented here in a later release.

## Planned public resources

- scans,
- reports,
- gas reports,
- proof records,
- webhook subscriptions,
- agent metadata.

When the API ships, every endpoint on this page should include method, path, auth, request schema, response schema, error codes, and example payloads.
